🔒 隐私政策

最后更新日期：2026年4月2日

Autoconfig（以下简称“我们”）重视并保护用户的隐私。本隐私政策说明我们如何收集、使用和保护你的个人信息。

1. 信息收集

我们仅收集为提供服务所必需的最少信息：

  • 账号信息 — 注册时提供的用户名和邮箱地址
  • 项目数据 — 你上传的静态网页文件
  • 访问数据 — 项目的访问次数（匿名统计）

2. 信息使用

你的信息仅用于：

  • 提供网页托管服务
  • 生成你的专属访问链接
  • 在社区广场展示公开项目（仅限你选择公开的项目）
  • 基础的访问统计

3. 信息保护

  • 密码经过 bcrypt 加密存储，即使数据库泄露也无法获取原文
  • 使用 JWT Token 进行身份认证，避免会话劫持
  • 私密项目仅通过直接链接访问，不会被搜索引擎索引

4. 数据删除

你可以随时：

  • 删除任何已部署的项目（文件将被永久删除）
  • 联系我们删除你的账号和所有相关数据

5. Cookie 使用

我们使用 Cookie 仅用于维持登录状态（JWT Token）。不使用任何第三方跟踪 Cookie。

6. 第三方服务

我们不与任何第三方共享你的个人信息或项目数据。

7. 政策更新

我们可能会不定期更新本隐私政策。重大变更时将在平台上通知用户。

8. 联系方式

如果你对隐私政策有任何疑问，请通过 GitHub 联系我们。
