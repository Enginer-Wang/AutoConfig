⚡ 关于 Autoconfig

Autoconfig 是一个专为开发者打造的轻量级静态网页托管平台。我们的使命是让网页发布变得极其简单 — 上传即上线。

🎯 我们的愿景

在当今复杂的部署流程中，很多开发者只是想快速分享一个 HTML 页面、展示一个个人简历或测试一个前端原型。传统的托管服务往往需要：

  • 配置 CI/CD 流水线
  • 设置服务器和域名
  • 处理 SSL 证书
  • 管理容器或云服务

Autoconfig 将这一切简化为三步：上传 → 部署 → 分享。

✨ 核心特性

  • 秒级部署 — ZIP 上传后自动解压、配置，几秒钟即可上线
  • 专属子域名 — 每个用户拥有 username.autoconfig.dev 的独立空间
  • 隐私优先 — 默认私密访问，只有你分享的人才能看到
  • 社区广场 — 选择公开的项目将展示在社区，获得更多曝光
  • 零配置 — 无需任何服务器知识，拖拽即部署

🛠️ 技术栈

Autoconfig 使用现代 Web 技术构建：

  • 后端 — Node.js + Express
  • 数据库 — SQLite (轻量、零配置)
  • 认证 — JWT Token + Cookie
  • 文件处理 — Multer + adm-zip
  • 前端 — 原生 HTML/CSS/JS (无框架依赖)
